PERSONAL PHOTOS — Placement Guide
==================================

Place the following three photos in this directory:

  sheikh-with-father.jpg   ← Photo of Sheikh Ammar with his father (natural, candid moment)
  sheikh-with-mbz.jpg      ← Photo of Sheikh Ammar with H.H. President Mohammed bin Zayed Al Nahyan

File format: JPG or WebP
Recommended resolution: 1200 × 900 px minimum (4:3 or 3:4 portrait)
Max file size: 3 MB per image (compress with TinyJPG or Squoosh if needed)

Once the files are placed here, the CollectorStory section on the homepage
will automatically display them. No code changes needed.

If you upload via the admin panel, note the generated URL and update:
  client/src/components/CollectorStory.tsx → the `content.panels[].image` paths
